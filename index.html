FEMRI by alexmbcm

An MRI simulator for unstructured meshes

This library is used to compute the MRI signal of an unstructured mesh using the Fourier Transform approximation to the Bloch equations.

Dependencies

vmtk-0.9 or vmtk-0.8 (with some file changes)

Install

Follow the instructions in: http://www.vmtk.org/Main/Installation to install vmtk. Then use github to install FEMRI.
